Assignment 2 - Flight Reservation Management System
March 27, 2022
Authors: Allyssa Preece, Anusone Soula, Aiden Kopec

This program is a graphical user interface for a flight reservation management system. It reads the required resource files in the res/ folder to operate.
All changes made within the application are saved into a random access binary file in res/.

To run the program, click on the assignment2.jar. 
For more infomation such as console logging, use the following command in the root directory: java -jar .\assignment2.jar.
